# Promediómetro
• Promediómetro es una calculadora simple para estimar tu promedio final en ramos USM. Permite ingresar   certámenes, controles, tareas y globales según las reglas de cada asignatura, mostrando al instante si   apruebas, si necesitas global o qué nota mínima debes sacar para salvar el ramo.
[Accede aquì](https://lucasfluxa.github.io/Promediometro/)
